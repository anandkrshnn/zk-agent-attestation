## 🚀 Getting Started

### For Developers: Start with Local Agent

**Try the personal-scale demo:**

```bash
git clone https://github.com/anandkrshnn/local-agent.git
cd local-agent
pip install -e .
ollama pull phi3:mini
local-agent serve
```

**Learn:** Permission broker pattern, cryptographic tokens, audit trails  
**Time:** 10 minutes to running
